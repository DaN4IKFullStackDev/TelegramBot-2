# TelegramBot-2
Игровой Telegram‑бот на C#. Основные возможности:  мини‑игра с тапами система начисления и отслеживания очков; таблица лидеров (/top); магазин бонусов (усиления, VIP‑статус, роль); админ‑панель с командами для управления пользователями и очками; роли пользователей (player, legend) и временные привилегии (VIP‑админ). Хранилище данных — SQLite.
